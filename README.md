# Sand — Particle Physics Sandbox

A lightweight browser-based particle sandbox where simple rules create surprisingly satisfying behavior. Pour sand, watch materials interact, and experiment with emergent physics in real time. Built as a standalone HTML experience with no install required.

## Features

- Real-time particle simulation
- Falling sand / granular physics
- Lightweight standalone build
- Runs directly in the browser
- No dependencies or setup required
- Smooth interactive rendering
- Experimental sandbox environment

## Demos

Open the included HTML file in any modern browser:

```bash
Sand _standalone_.html
```

Or 

Try the sandbox here:

👉 https://sand.xozy.app

## Controls

| Action | Control |
|---|---|
| Place particles | Left Click |
| Erase particles | Right Click / Alternate Tool |
| Draw continuously | Click + Drag |
| Experiment | Try mixing materials and shapes |

## Tech

Built with:

- HTML5
- JavaScript
- Canvas rendering
- Custom particle simulation logic

## Why I Made This

This project started as a small experiment in emergent behavior and physics simulation. Even with simple movement rules, particle systems can create surprisingly organic and satisfying interactions.
